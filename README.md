# Serverless Framework

### Deployment - Development

```
$ npm run deploy:dev
```

### Deployment - Production

```
$ npm run deploy:prod
```

### Test Lamdas Offline

```
$ npm run start:dev
```

### Swagger UI

```
$ npm run start:swagger
```

### Dev - Start Lamdas and Swagger Concurrently

```
$ npm run dev
```
